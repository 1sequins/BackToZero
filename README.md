## BackToZero  


**BACK2ZERO** 是[2015 indieplay独立游戏嘉年华Game Jam第一名](http://www.sonkwo.com/articles/1776)作品，制作人@沉默之沙 在[开发日志](http://v.qq.com/x/page/a0160hz6rew.html)中讲述了游戏设计思路，这里用Unity还原该游戏，游戏名字为 `BackToZero`。
